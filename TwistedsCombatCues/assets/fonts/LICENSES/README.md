# Bundled font licenses

License texts for the **open-licensed** fonts in `assets/fonts/`. Each font ships with
the full text of its license, as required by the SIL Open Font License and the Ubuntu
Font Licence.

| Font file(s) | License | Text |
| --- | --- | --- |
| `FiraSans Bold.ttf`, `FiraSans Light.ttf`, `FiraSans Medium.ttf` | SIL OFL 1.1 | `OFL-FiraSans.txt` |
| `Poppins.ttf` | SIL OFL 1.1 | `OFL-Poppins.txt` |
| `Barlow Condensed.ttf` | SIL OFL 1.1 | `OFL-BarlowCondensed.txt` |
| `Changa.ttf` | SIL OFL 1.1 | `OFL-Changa.txt` |
| `Cinzel Decorative.ttf` | SIL OFL 1.1 | `OFL-CinzelDecorative.txt` |
| `Exo.otf` | SIL OFL 1.1 | `OFL-Exo.txt` |
| `Russo One.ttf` | SIL OFL 1.1 | `OFL-RussoOne.txt` |
| `Ubuntu.ttf` | Ubuntu Font Licence 1.0 | `UFL-Ubuntu.txt` |

**Not covered here.** The other fonts in `assets/fonts/` are *not* open-licensed and are
not covered by these files - see `../../THIRD_PARTY_NOTICES.md`. The proprietary ones
(`Arial *`, `Gotham *`, `Avant Garde Naowh`) are **not redistributable** and should be
removed before a public release.
